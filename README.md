## Инструкция по работе    

### Где брать задания

1. Репозитории https://github.com/Finuniversity/WorkshopPython.git    
2. Папка Task    
3. Файл Task.md    

### Как сдавать задания    

В своем репозитории создаем папку с именем задания (Пример: 1-Task, 2-Task и т.д.) и туда размещаем файлы с кодом
